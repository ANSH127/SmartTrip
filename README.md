
# SmartTrip

A mobile application that helps you track your expenses on a trip.

## Run Locally

Clone the project

```bash
  git clone https://github.com/the-error4-0-4/SmartTrip.git
```

Go to the project directory

```bash
  cd SmartTrip
```

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npx expo start
```
